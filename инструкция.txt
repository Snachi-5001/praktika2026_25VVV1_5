Инструкция для студентов:

Для начала клонируйте репозиторий к себе на компьютер командой (git clone https://github.com/Snachi-5001/praktika2026_25VVV1_5.git).

Перед тем как начать писать код, всегда создавайте отдельную ветку для своей задачи командой git checkout -b feature/имя_вашей_задачи. Это нужно, чтобы не сломать основной код в ветке main.

После внесения изменений в файлы сохраните их локально, добавив в индекс (git add .) и сделав коммит (git commit -m "Ваше описание изменений").

Отправьте свою ветку на сервер командой (git push origin feature/имя_вашей_задачи).

Зайдите на страницу репозитория на GitHub, там появится кнопка Compare & pull request. Нажмите её, проверьте, что всё верно, и создайте Pull Request.

После этого я проверю ваш код, и если всё хорошо, мы вольем его в общую ветку. Не удаляйте свои ветки до тех пор, пока я не подтвержу, что изменения успешно приняты.
